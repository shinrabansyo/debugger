### CLI

```
$ cargo run --bin sb_dbg_cli -- --inst examples/helloworld/helloworld.asm
Hello world!
```

### TUI

```
$ cargo run --bin sb_dbg_tui -- --inst examples/helloworld/helloworld.asm
```
